支付宝Java SDK
===============
官方的Demo代码效率不够高，所以自己基于spring boot写了一个demo。

V1.0
----
最新版PC的扫码支付


使用前的配置
===============
https://docs.open.alipay.com/291/106103/

https://docs.open.alipay.com/270/alipay.trade.page.pay/

----
直接启动项目 AlipayApplication
